# Study Bridge AI

## 📌 Description
Study Bridge AI is a simple AI-based web app that helps students get answers to study-related questions.

## 🚀 Features
- Ask questions
- Get instant answers
- Uses local dataset

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download project
2. Open index.html
3. Ask questions
